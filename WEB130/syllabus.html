Course Description: This hands-on course covers basic to advanced features of Macromedia Dreamweaver MX. Students will learn to plan, build, upload, and maintain a professional website using Macromedia Dreamweaver MX.
Course Outcomes: Upon successful completion of WEB130, the student will be able to design and produce a professional website with confidence including:

• Functional viewability in the latest browsers.
• The understanding of file names, formats, extensions and how they work together.
• Proficiency in using graphics that successfully utilizes the ratio of maximum colors to minimum file size.
• Implementation of text and graphic combinations which provide a clear and readable design geared toward the target audience
• The understanding of functional buttons and links.
• The basic understanding of implementing frames in your page.
• The understanding of how to upload and manage files to the server
• Understanding the basics of cleaning up and managing code for optimal performance of the site.
• The understanding of basic JavaScript Behaviors.
• The basic understanding of “Form Mail.”
Required Text: Macromedia Dreamweaver MX for Windows and Macintosh: Visual QuickStart Guide, by J. Tarin Towers

HTML 4 for the World Wide Web: Visual QuickStart Guide, by Elizabeth Castro
